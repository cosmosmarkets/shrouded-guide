# Shrouded Guide

Interactive onboarding site for Shrouded Development — developer and recruiter guides.

## Stack

- Vite + React + TypeScript
- React Router
- Lenis (smooth scroll)
- GSAP + ScrollTrigger

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Deploy output is `dist/` (configured for Vercel SPA rewrites).
